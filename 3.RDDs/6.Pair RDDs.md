# Pair RDDs in Apache Spark – Complete Structured Notes

# Introduction

So far, we have worked with **Normal RDDs**, where data is stored as a collection of elements.

Example:

```text
['Spark', 'Hadoop', 'Spark', 'Python']
```

However, in real-world Big Data processing, we often need to store data as **Key-Value Pairs**.

For this purpose, Spark provides **Pair RDDs (Paired RDDs)**.

---

# What is a Pair RDD?

## Definition

A **Pair RDD** is a special type of RDD where each record is stored as a **(Key, Value)** pair.

### Syntax

```text
(Key, Value)
```

### Example

```text
('HR', 1000)

('Finance', 2000)

('IT', 3000)
```

---

# Visual Representation

Normal RDD:

```text
HR
Finance
IT
HR
IT
```

Pair RDD:

```text
(HR, 1)
(Finance, 1)
(IT, 1)
(HR, 1)
(IT, 1)
```

---

# Why Pair RDDs?

Pair RDDs allow Spark to:

* Process data based on keys
* Aggregate values efficiently
* Perform joins
* Perform grouping
* Perform counting

---

# Advantages of Pair RDDs

### Parallel Processing

Operations can be performed independently on each key.

### Aggregation

Easy counting and summation.

### Data Analysis

Useful for:

* Word Count
* Sales Analysis
* Customer Analysis
* Log Processing

### Distributed Computation

Keys can be processed across multiple worker nodes.

---

# Creating a Pair RDD

## Example

### Normal RDD

```python
words = sc.parallelize([
    "Spark",
    "Hadoop",
    "Spark",
    "Python"
])
```

Output:

```text
Spark
Hadoop
Spark
Python
```

---

### Convert to Pair RDD

```python
pair_rdd = words.map(
    lambda x:(x,1)
)
```

Output:

```text
(Spark,1)

(Hadoop,1)

(Spark,1)

(Python,1)
```

# Real-World Example: Word Count

Word Count is one of the most famous examples used to explain Pair RDDs.

---

# Input Data

```text
Apache Spark
Apache Hadoop
```

---

# Step 1: Create RDD

```python
data = [
"Apache Spark",
"Apache Hadoop"
]

rdd = sc.parallelize(data)
```

---

# Step 2: Tokenize Using flatMap()

```python
words = rdd.flatMap(
    lambda x:x.split()
)
```

Output:

```text
Apache
Spark
Apache
Hadoop
```

---

# Step 3: Convert into Pair RDD

```python
pair_rdd = words.map(
    lambda word:(word,1)
)
```

Output:

```text
(Apache,1)

(Spark,1)

(Apache,1)

(Hadoop,1)
```

---

# Step 4: Aggregate Values

```python
word_count = pair_rdd.reduceByKey(
    lambda x,y:x+y
)
```

Output:

```text
(Apache,2)

(Spark,1)

(Hadoop,1)
```

# Common Pair RDD Transformations

---

# 1. reduceByKey()

## Definition

Combines values having the same key.

---

## Example

Input:

```text
(HR,1000)

(HR,2000)

(IT,3000)
```

Code:

```python
rdd.reduceByKey(
lambda x,y:x+y
)
```

Output:

```text
(HR,3000)

(IT,3000)
```

---

# 2. groupByKey()

## Definition

Groups all values belonging to the same key.

---

Input:

```text
(HR,1000)

(HR,2000)

(IT,3000)
```

Output:

```text
HR → [1000,2000]

IT → [3000]
```

---

# 3. sortByKey()

## Definition

Sorts records based on key.

---

Input:

```text
(C,100)

(A,200)

(B,300)
```

Output:

```text
(A,200)

(B,300)

(C,100)
```

---

# 4. join()

## Definition

Joins two Pair RDDs using common keys.

---

RDD1

```text
(101,John)

(102,Sarah)
```

RDD2

```text
(101,HR)

(102,IT)
```

Output:

```text
(101,(John,HR))

(102,(Sarah,IT))
```

---

# Common Pair RDD Actions

---

# 1. countByKey()

## Definition

Counts number of occurrences for each key.

---

Input:

```text
(HR,1000)

(HR,2000)

(IT,3000)
```

Output:

```text
HR → 2

IT → 1
```

---

# 2. collect()

Returns all records.

---

# 3. count()

Returns total records.

---

# Pair RDD vs Normal RDD

| Feature         | Normal RDD   | Pair RDD       |
| --------------- | ------------ | -------------- |
| Structure       | Single Value | Key-Value Pair |
| Aggregation     | Difficult    | Easy           |
| Join Support    | No           | Yes            |
| Grouping        | Limited      | Excellent      |
| Counting by Key | Not Possible | Possible       |

# Complete Word Count Program

```python
from pyspark import SparkContext

sc = SparkContext()

data = [
    "Apache Spark",
    "Apache Hadoop",
    "Apache Spark"
]

rdd = sc.parallelize(data)

words = rdd.flatMap(
    lambda x:x.split()
)

pair_rdd = words.map(
    lambda x:(x,1)
)

result = pair_rdd.reduceByKey(
    lambda x,y:x+y
)

print(result.collect())
```

Output:

```text
[
 ('Apache',3),
 ('Spark',2),
 ('Hadoop',1)
]
```

# Memory Trick

### Pair RDD = KVA

**K → Key**

**V → Value**

**A → Aggregation**

Remember:

> **Pair RDDs are designed for key-based processing and aggregation in distributed environments.**

This is one of the most important Spark concepts because most real-world Spark jobs (Word Count, Sales Analysis, Log Analytics, Joins, Aggregations) are built using **Pair RDD operations**. 🚀
