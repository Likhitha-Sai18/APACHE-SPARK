# View Data Inside Partitions

Use:

```python

```

### What is glom()?

`glom()` groups elements partition-wise.

Example:

```python
[[1], [2], [3], [4], [5], [], [], []]
```

Notice:

âś” Data distributed across partitions

âś” Some partitions may be empty

# Why Empty Partitions?

If:

```text
Data = 10 records
Partitions = 16
```

Not every partition will receive data.

```text
Partition 1 â†’ [1]
Partition 2 â†’ [2]
...
Partition 10 â†’ [10]
Partition 11 â†’ []
```

# Creating RDD with Custom Partitions

Suppose:

```python
countries = [
    "USA",
    "India",
    "Canada",
    "Australia",
    "Germany"
]
```

Create RDD with 2 partitions:

```python
rdd = sc.parallelize(countries, numSlices=2)
```

or

```python
rdd = sc.parallelize(countries, 2)
```

---

# Verify Partitions

```python
rdd.getNumPartitions()
```

Output:

```python
2
```

---

# Check Partition Data

```python
rdd.glom().collect()
```

Output:

```python

 ['Canada', 'Australia', 'Germany']
]
```

Visualization:

```text
Germany
```

# Method 2: Create RDD from External File

Spark can read files directly.

Example:

```python

```

---

# Specify Minimum Partitions

```python
rdd = sc.textFile(
    "movies.csv",

)
```

Spark creates at least:

# Important Functions

| Function             | Purpose                    |
| -------------------- | -------------------------- |
| `parallelize()`      | Create RDD from collection |
| `textFile()`         | Create RDD from file       |
| `getNumPartitions()` | Number of partitions       |
| `glom()`             | View data partition-wise   |
| `collect()`          | Bring data to driver       |

---

# Difference: numSlices vs minPartitions

| Parameter       | Used With       | Purpose                     |
| --------------- | --------------- | --------------------------- |
| `numSlices`     | `parallelize()` | Exact partitions desired    |
| `minPartitions` | `textFile()`    | Minimum partitions required |

Example:

```python
sc.parallelize(data, 4)
```

Creates:

```text
Exactly 4 partitions
```

---

```python
sc.textFile("data.csv", 4)
```

Creates:

```text
At least 4 partitions
```
