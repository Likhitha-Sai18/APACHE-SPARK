# Types of Machine Learning Problems (Simple & Easy Notes)

Machine Learning problems are mainly divided into **two types**:

1. **Supervised Learning**
2. **Unsupervised Learning**

---

# 1. Supervised Learning

## Definition

In **Supervised Learning**, the computer learns from **labeled data**.

A **label** means the correct answer is already available in the dataset.

The model learns the relationship between the **input features** and the **target output**, then predicts outputs for new data.

### Simple Formula

```
Input Data + Correct Answer (Label)
            ↓
      Train Model
            ↓
Predict Output for New Data
```

---

## Example Dataset

| Weight (kg) | Height (cm) | Obesity |
| ----------- | ----------- | ------- |
| 55          | 160         | No      |
| 90          | 170         | Yes     |
| 70          | 168         | No      |
| 110         | 180         | Yes     |

Here:

* **Weight** → Input Feature
* **Height** → Input Feature
* **Obesity** → Target (Label)

The model learns:

> Weight + Height → Predict Obesity

When a new person's weight and height are given, the model predicts whether the person is obese.

---

# How Supervised Learning Works

### Step 1

Collect labeled data.

```
Features + Labels
```

↓

### Step 2

Choose a Machine Learning algorithm.

↓

### Step 3

Train the model.

↓

### Step 4

Provide new data.

↓

### Step 5

Model predicts the output.

# Types of Supervised Learning

There are **two major types**:

## 1. Regression

### Definition

Regression is used when the output is a **continuous numerical value**.

Continuous values can take any number within a range.

### Examples

* House price prediction
* Stock price prediction
* Temperature prediction
* Rainfall prediction
* Sales forecasting

### Example

Input:

```
Area = 1200 sq.ft
Bedrooms = 2
```

Output:

```
House Price = ₹45,50,000
```

The output is a number, so this is a **Regression problem**.

---

## 2. Classification

### Definition

Classification predicts **categories or classes**, not numbers.

### Examples

* Spam or Not Spam
* Pass or Fail
* Disease or No Disease
* Fraud or Genuine Transaction
* Cat or Dog Image

### Example

Input:

```
Email Text
```

Output:

```
Spam
```

or

```
Not Spam
```

Since the output belongs to predefined categories, this is a **Classification problem**.

---

# Regression vs Classification

| Regression           | Classification          |
| -------------------- | ----------------------- |
| Predicts numbers     | Predicts categories     |
| Continuous output    | Categorical output      |
| Example: House price | Example: Spam detection |
| Example: Temperature | Example: Pass/Fail      |

---

# 2. Unsupervised Learning

## Definition

In **Unsupervised Learning**, the computer learns from **unlabeled data**.

There are **no correct answers (labels)** provided.

The model tries to discover hidden patterns, similarities, or structures on its own.

### Simple Formula

```
Input Data (No Labels)
          ↓
Model Finds Patterns
          ↓
Groups Similar Data
```

---

## Example

Imagine you have pictures of different cartoon characters, but none of them have names.

The model doesn't know:

* Which one is Mickey Mouse
* Which one is Donald Duck
* Which one is Tom
* Which one is Jerry

Instead, it groups similar-looking cartoons together based on features like shape, color, and size.

This is **Unsupervised Learning**.

---

# Most Common Unsupervised Learning Problem

## Clustering

### Definition

Clustering groups similar data points into the same cluster without using labels.

### Example

A shopping website wants to divide customers into groups based on purchasing behavior.

The model may automatically create groups like:

* High spenders
* Medium spenders
* Budget shoppers
* Frequent buyers
* Occasional buyers

No labels are given beforehand—the model finds these groups itself.

---

# Supervised vs Unsupervised Learning

| Supervised Learning                  | Unsupervised Learning     |
| ------------------------------------ | ------------------------- |
| Uses labeled data                    | Uses unlabeled data       |
| Learns from correct answers          | Learns patterns by itself |
| Predicts outputs                     | Finds hidden patterns     |
| Has input and target                 | Has only input data       |
| Examples: Regression, Classification | Example: Clustering       |

* Learns from examples
* Predicts known outputs
* Types:

  * **Regression** → Continuous values
  * **Classification** → Categories

### Unsupervised Learning

* Uses **unlabeled data**
* Finds hidden patterns automatically

---

# Easy Way to Remember

**Supervised Learning = Teacher + Answer Sheet**

* The teacher provides questions and correct answers.
* The student learns from examples.

**Unsupervised Learning = Self-Learning**

* No teacher and no answer sheet.
* The student groups similar things by observing patterns.

This analogy captures the core difference:

* **Supervised Learning learns from labeled examples.**
* **Unsupervised Learning discovers patterns without labels.**
