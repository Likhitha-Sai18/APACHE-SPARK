# Step 1: Import Required Libraries

```python
from pyspark.sql import SparkSession
from pyspark.sql import types as TP
```

---

# Step 2: Create SparkSession

```python
spark = SparkSession.builder \
        .appName("HealthcareAnalytics") \
        .getOrCreate()
```

---

# Step 3: Define Schema

Since we know the structure of the data, we define it explicitly.

Example:

```python
schema = TP.StructType([
    TP.StructField("case_id",TP.IntegerType(),True),
    TP.StructField("Hospital_code",TP.IntegerType(),True),
    TP.StructField("Department",TP.StringType(),True),
    TP.StructField("Age",TP.StringType(),True)
])
```

---

# Why Define Schema?

Benefits:

✅ Better Performance

✅ Better Data Validation

✅ Avoids Schema Inference Cost

✅ Recommended for Large Datasets

---

# Step 4: Load CSV Data

Since the dataset already contains column headers:

```python
df = spark.read.csv(
    "healthcare.csv",
    header=True,
    schema=schema
)
```

Used to retrieve specific columns.

## Syntax

# Drop Multiple Columns

---

## Method

Create a list:

```python
cols_to_drop = [
    "Age",
    "Department",
    "Visitors_with_Patient"
]
```

---

Apply:

```python
new_df = df.drop(*cols_to_drop)
```

---

# Why Asterisk (*)?

Python unpacks list elements.

---

Without:

```python
drop(list)
```

With:

```python
drop(*list)
```

Equivalent to:

```python
drop(
 "Age",
 "Department",
 "Visitors_with_Patient"
)
```

# Filtering Rows

Often we need only specific records.

Use:

```python
where()
```

or

```python
filter()
```

Both are identical.

---

# where()

Used to select rows satisfying a condition.

---

## Syntax

```python
df.where(condition)
```

---

# Example 1

Retrieve records where:

```text
Hospital_code = 8
```

---

```python
sample_data.where(
    sample_data.Hospital_code == 8
).show()
```

---

## Output

| Hospital_code |
| ------------- |
| 8             |
| 8             |
| 8             |

Only matching records displayed.

---

# Visualization

```text
All Records
      ↓
Hospital_code == 8
      ↓
Filtered Records
```

---

# Multiple Conditions

Suppose we need:

```text
Hospital_code = 8

AND

Department = Radiology
```

---

## Syntax

```python
sample_data.where(
    (sample_data.Hospital_code == 8) &
    (sample_data.Department == "Radiology")
)
```

---

## Output

| Hospital_code | Department |
| ------------- | ---------- |
| 8             | Radiology  |
| 8             | Radiology  |

---

# Important Note

For multiple conditions:

Use:

```python
&
```

for AND

---

Use:

```python
|
```

for OR

---

Example:

```python
df.where(
    (df.Age == "41-50") |
    (df.Age == "51-60")
)
```

# Memory Trick

```text
SELECT → Keep Columns

DROP → Remove Columns

WHERE → Keep Rows
```

This is the **foundation of DataFrame manipulation**, and these operations are used in almost every Spark ETL pipeline. 🚀
